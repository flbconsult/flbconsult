## 👤 À propos

Je suis **CIO / CTO** avec **25+ ans d'expérience** dans le pilotage de la transformation digitale d'organisations critiques (santé, finance, télécom, SaaS). Mon focus actuel — et l'objet de ce portfolio — porte sur **l'intégration systémique de l'IA générative** dans les architectures d'entreprise, les enjeux de **cybersécurité** qu'elle introduit, et les arbitrages de **souveraineté cloud** qui en résultent.

Missions récentes : **CTO chez KHOME (Groupe Crédit Agricole)** — transformation AI-Native pour la valorisation et le scoring d'actifs immobiliers ; **CTO chez People First Technologies** — industrialisation RH × LLM/RAG ; **Directeur Exécutif AP-HP** — programme 30M€, continuité de service après perte des deux datacenters ; **CTO chez Vocalcom** — Move2Cloud AWS (570 clients), certification ISO 27001 ; **CTO chez Vivoka** — 6 brevets NLP/NLU déposés, chaire industrielle INRIA.
